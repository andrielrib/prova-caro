HTML

HTML é a linguagem de marcação usada para estruturar páginas web.

  • <html> - Define um documento HTML
  • <head> - Contém metadados, título e links
  • <title> - Define o título da aba
  • <body> - Define o corpo da página
  • <h1> até <h6> - Títulos
  • <p> - Parágrafos
  • <a href="URL"> - Links
  • <img src="imagem.jpg" alt="Descrição"> - Imagens
  • <ul> e <ol> - Listas
  • <table> - Tabelas
  • <div> - Agrupamento
  • <span> - Texto inline

CSS

CSS é usado para estilizar páginas HTML.

  • color - Cor do texto
  • background-color - Cor do fundo
  • font-size - Tamanho da fonte
  • margin / padding - Espaçamento
  • display: flex; - Layout flexível
  • width / height - Largura e altura

JavaScript

JavaScript é uma linguagem de programação usada para interatividade em páginas web.

  • document.getElementById("id") - Seleciona um elemento pelo ID
  • document.querySelector(".classe") - Seleciona um elemento pela classe
  • addEventListener("click", function) - Adiciona um evento de clique
  • let x = 10; - Declaração de variável
  • function minhaFuncao() {} - Declaração de função
  • console.log("Mensagem"); - Exibe uma mensagem no console

Git & GitHub

Git é um sistema de controle de versões.

  • git init - Inicia um repositório
  • git clone URL - Clona um repositório
  • git add . - Adiciona mudanças
  • git commit -m "mensagem" - Salva mudanças
  • git push origin branch - Envia mudanças
  • git pull origin branch - Atualiza repositório
